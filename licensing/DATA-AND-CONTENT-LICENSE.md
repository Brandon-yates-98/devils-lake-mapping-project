## 2. OpenStreetMap-derived data, ODbL 1.0 (cannot be NC)

Any data derived from **OpenStreetMap** remains under the **Open Database
License (ODbL) 1.0** and is **© OpenStreetMap contributors**. ODbL has no
noncommercial option, so the project's NC restriction **cannot** apply to this
layer. Keep OSM-derived data **separable** from original data so the NC terms
and the ODbL terms each apply only to their own layer.

<https://opendatacommons.org/licenses/odbl/1-0/>

## 3. Facts are not owned

Factual data, campsite numbers and positions, road/loop geometry, amenity
locations, is not protected by copyright (*Feist v. Rural*). The licenses above
cover original **expression and compilation**, not the underlying facts.

## 4. Upstream contributions (OSM / OpenBeta)

Data the project contributes **to** OpenStreetMap or OpenBeta is provided under
**those projects' terms** (ODbL / CC-BY-SA / CC0), not under CC BY-NC. That
contributed slice is a deliberate gift and is not bound by the NC restriction.

> Do **not** upstream data traced from copyrighted sources (e.g., the WI DNR
> campground map) into OSM, OSM prohibits imports from copyrighted sources
> without permission. Upstream only survey/GPS-verified facts.
